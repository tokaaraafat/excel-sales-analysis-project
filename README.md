# Excel Sales Analysis Dashboard

## Project Overview

This project analyzes sales and operational data using Microsoft Excel to identify trends and track key performance indicators (KPIs).
The dataset was cleaned and transformed using Power Query, and interactive dashboards were built using Pivot Tables and advanced Excel formulas to generate business insights.

## Tools Used

* Microsoft Excel
* Power Query (Data Cleaning & Transformation)
* Pivot Tables
* Advanced Excel Formulas

## Project Tasks

* Cleaned and transformed raw sales data using Power Query
* Structured datasets for analysis and reporting
* Built Pivot Table dashboards to track KPIs
* Created automated calculations using Excel formulas
* Analyzed sales performance trends

## Key Insights

* Identified monthly sales trends and performance patterns
* Highlighted top-performing products and categories
* Built KPI dashboards for business performance tracking
* Improved reporting efficiency through automated calculations

* `sales_dashboard.xlsx` → Excel analysis, Power Query transformations, and dashboards
